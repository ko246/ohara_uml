# ユースケース名
在庫確認

# ユースケース概要
管理者が注文管理システムで商品の在庫状況を確認する。

# アクター
倉庫業者

# 事前条件
- 管理者がログイン状態であること

# 事後条件
- 管理者が特定商品の詳細在庫情報を閲覧、検索ができる

# 基本フロー
1. 管理者はメニュー画面から「在庫確認」機能を選択する
2. システムは商品一覧と在庫数量を表示する
    1. 在庫情報が取得できない場合、代替フロー1へ
3. 管理者は必要に応じて特定商品の詳細在庫情報を閲覧、検索が可能

# 代替フロー
1. システムはエラーメッセージを表示する


# 関連情報
- 管理者ID、商品ID、商品名、在庫数量
