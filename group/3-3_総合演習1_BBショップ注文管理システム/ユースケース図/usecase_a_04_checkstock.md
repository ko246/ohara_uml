# ユースケース名
在庫確認

# ユースケース概要
管理者が注文管理システムで商品の在庫状況を確認する。

# アクター
倉庫業者

# 事前条件
- 管理者がログイン状態であること
- システムはメニュー画面を表示している

# 事後条件
- 管理者が特定商品の詳細在庫情報を閲覧、検索ができる
- システムは商品の詳細在庫情報画面を表示する

# 基本フロー
1. システムはメニュー画面を表示する
2. 管理者はメニュー画面から「在庫確認」機能を選択する
3. システムは商品一覧画面を表示する
4. 管理者は必要に応じて特定商品の詳細在庫情報を閲覧、検索が可能
    1. 在庫情報が取得できない場合、代替フロー1へ

# 代替フロー
1. システムはエラーメッセージを表示し、基本フロー3へ


# 関連情報
- 社員ID、商品ID、商品名、在庫数量
